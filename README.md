## DER Interconnection Study & Compliance Analysis (FortisAlberta Standards)

**Status:** Work in Progress  

**Tools:** PSS®E, Microsoft Excel  

### Overview
This project focuses on performing a Distributed Energy Resource (DER) interconnection study to evaluate the integration of a generator into a utility network, ensuring compliance with FortisAlberta and IEEE standards.

### Current Scope
- Conducting DER interconnection study using **PSS®E**  
- Performing **short-circuit** and **effective grounding studies**  
- Evaluating **protection system performance** under fault conditions  
- Assessing system impacts on **reliability and grid integration**  

### Planned Work
- Complete **protection coordination and relay settings validation**  
- Finalize **Single Line Diagram (SLD)** with mitigation measures  
- Develop **SCADA/DNP3 point list** for monitoring and control  
- Prepare detailed **technical report** aligned with utility standards  

### Standards & References
- FortisAlberta DER Standards (DER-02 Series)  
- IEEE 1547 – DER Interconnection and Interoperability  

### Expected Outcomes
- Verified compliance with utility interconnection requirements  
- Improved system reliability through mitigation strategies  
- Industry-ready documentation for real-world utility applications
